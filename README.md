# Recommendation System

## 📌 Overview
Built a machine learning-based recommendation system to suggest personalized content based on user preferences and interaction patterns.

## ⚙️ Tech Stack
Python, Pandas, Scikit-learn, Streamlit

## 🚀 Features
- User-item based recommendation
- Similarity-based ranking
- Real-time suggestions using Streamlit

## 📊 How It Works
- Data preprocessing and user-item matrix creation
- Similarity calculation using cosine similarity
- Ranking items based on relevance

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app/app.py

## 📈 Results
- Improved recommendation relevance using similarity-based methods
